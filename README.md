# Inventory Management System

## Overview
Java-based Inventory Management System using advanced Collection Framework concepts.

## Features
- Unique product management using HashSet
- Sorted views using TreeSet & Comparator
- Undo functionality using Stack
- Low stock alerts using Queue
- Transaction history using LinkedList

## Technologies
- Java
- Collections Framework
- OOP Principles

## How to Run
1. Compile all `.java` files
2. Run `Main.java`
3. Use console menu

## Concepts Used
- Comparable & Comparator
- Generics
- Data structure optimization
- Time complexity awareness
